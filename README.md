## Setup

Install [Node.js](https://nodejs.org/en/) and [Yarn](https://yarnpkg.com/)

Clone this repository

```sh
git clone https://github.com/tmvdl/speed.git
cd speed
```

Install the dependencies

```sh
yarn
```

Run the project

```sh
yarn start
```

View on browser

http://localhost:4200/
